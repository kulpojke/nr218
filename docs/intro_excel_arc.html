File Organization and Computer Hygiene

Be kind to your future self - keep your files organized:

  • Use directories (a.k.a folders)
  • Use descriptive names:
    • e.g. slo_county_26910.geojson not county.geojson
  • Don’t put spaces in file or directory names (see this)

Tutorial: Export QGIS Data to Excel

  • Open the Swanton dataset in QGIS.
  • Right-click sprLandUse → Export → MS Office Open XML spreadsheet (XLSX).
  • Save the file to an accessible location (e.g., Desktop or OneDrive).

Excel Online: Load GIS Data

  • Visit office.com and sign in with CP credentials.
  • Explore apps → open Excel → File → Open → Open files from this device.
  • Select the exported XLSX to view GIS attributes in Excel Online.

Excel Basics: Sort & Filter

  • Use the funnel icon to enable filters on each column header.
  • Sort ascending/descending and filter categories to explore the dataset.
  • Observe how filtering impacts downstream calculations.

Excel Analysis: Summaries & Charts

  • Build mini-tables to compute metrics (e.g., % land area by use).
  • Highlight data → Insert → Chart to visualize (pie, column, etc.).
  • Format data labels and legends for clarity.

Excel Analysis: Pivot Tables

  • Select a categorical field (e.g., LUType) and a numeric field (Acres).
  • Data tab → Analyze Data → Insert Pivot Table.
  • Inspect the generated sheet to see totals by category; experiment with other stats.

Excel Analysis: Histograms

  • Highlight the Acres column → Insert → Chart → Histogram.
  • Adjust bin width to reveal distribution patterns and outliers.

Excel Analysis: Scatterplots & Trendlines

  • Select paired fields (e.g., height vs diameter) → Insert → Scatter.
  • Add trendlines and display equation plus R² to assess relationships.
  • When values cluster along an axis, use logarithmic scales to spread the data.

New Material: ArcGIS Pro Overview

  • ESRI’s enterprise GIS platform; widely used in government and large nonprofits.
  • Feature-rich and user-friendly but proprietary, Windows-only, and licensed.
  • Fewer plugins/community resources compared to QGIS; plan support accordingly.

Tutorial: ArcGIS Pro Setup

  • Launch ArcGIS Pro from the Start menu.
  • Download SwantonGIS.gdb.zip from Canvas and extract it.
  • In ArcGIS Pro: Map → Add Data → browse to the unzipped GDB.
  • Drag layers from the geodatabase into the map.

ArcGIS Pro: Symbology

  • Category symbology: right-click layer (e.g., sprVegetation) → Symbology → Unique Values → choose field (Vegtype1).
  • Graduated styling: experiment with numerical fields (sprSoils):
    • Graduated Colors with KFACTOR.
    • Graduated Symbols with Depth.
    • Proportional Symbols with SlopeMax.

ArcGIS Pro: Layouts & Export

  • Insert → New Layout → 8.5 x 11 Letter; set scale (e.g., 1:55,000).
  • Add map frame, north arrow, scale bar, legend, and grid coordinates.
  • Share the finished map via Share → Export Map, setting filename and path.
